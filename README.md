# Tattoo-Design-GUI

 Tattoo Flash Index (HTML GUI)

This is a simple, single‑page HTML interface designed to simulate a tattoo studio workflow. It includes client intake fields, an AI‑style design prompt box, a generated design preview area, size/estimate selectors, and print‑ready consent/consult buttons. Everything runs locally in the browser — no backend or external setup required.

---

## 🔍 What This Project Is

A front‑end demo that shows how a tattoo shop could use a digital form to:

- Collect client information  
- Enter tattoo concept ideas  
- Display generated design previews (static placeholders for demo)  
- Select size and view estimated time/price  
- Print a consent form or consult ticket  

This project is **HTML‑only** and meant as a visual/UX showcase.

## 🛠️ How to Use

1. Download the repository  
2. Open `index.html` in any browser  
3. Fill out the form fields  
4. Enter a tattoo concept  
5. View the design preview section  
6. Select a size to see estimated time/price  
7. Use the print buttons to generate forms  

No installation, no dependencies — it’s all in one file.
